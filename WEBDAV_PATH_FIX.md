# Исправление проблемы с путями WebDAV

## Проблема

При синхронизации создавалась одна папка с именем `test_manager\incoming` вместо правильной структуры `test_manager/incoming/`.

**Ошибка:** `Failed to upload: 409 (Conflict)`

## Причина

WebDAV использует URL пути с прямыми слешами `/`, а Windows использует обратные слеши `\`. При передаче Windows путей в WebDAV они интерпретировались как одно имя папки.

## Исправление

Обновлен файл `src/sync/real_webdav_client.py`:

1. **Все методы теперь конвертируют пути:**
   ```python
   remote_path = remote_path.replace('\\', '/')
   ```

2. **Рекурсивное создание папок:**
   - Метод `create_directory()` теперь создает все родительские папки
   - Разбивает путь на части: `test_manager/incoming` → `['test_manager', 'incoming']`
   - Создает каждую папку последовательно

3. **Исправленные методы:**
   - `upload_file()` - конвертирует пути перед загрузкой
   - `download_file()` - конвертирует пути перед скачиванием
   - `create_directory()` - рекурсивное создание
   - `delete_file()` - конвертирует пути
   - `move_file()` - конвертирует оба пути
   - `file_exists()` - конвертирует путь
   - `list_files()` - конвертирует путь

## Если проблема осталась

### Проверка 1: Очистите старые папки на WebDAV
Удалите неправильно созданные папки с именами типа `test_manager\incoming`.

### Проверка 2: Проверьте URL WebDAV
URL должен заканчиваться на `/`:
- ✅ Правильно: `https://webdav.server.com/dav/`
- ❌ Неправильно: `https://webdav.server.com/dav`

### Проверка 3: Проверьте права доступа
Убедитесь, что у пользователя WebDAV есть права на создание папок.

### Проверка 4: Посмотрите детальный лог

## Дополнительная информация

**Код 409 (Conflict)** означает:
- Попытка создать ресурс, который уже существует
- Попытка создать папку, когда родительская папка не существует
- Попытка загрузить файл в несуществующую папку

**Решение в коде:**
```python
# Было (неправильно):
url = f"{self.url}/{remote_path}"  # test_manager\incoming

# Стало (правильно):
remote_path = remote_path.replace('\\', '/')  # test_manager/incoming
url = f"{self.url}/{remote_path}"
```
