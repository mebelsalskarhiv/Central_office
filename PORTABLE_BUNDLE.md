# OrderManager Central Office - Portable Bundle

## Описание
Portable версия центрального офиса OrderManager для установки на любой компьютер без установки Python.

## Системные требования
- Windows 10 или новее (64-bit)
- Не требуется установка Python
- ~700 MB свободного места на диске
- Интернет для загрузки карт OpenStreetMap (опционально)

## Установка

### Вариант 1: ZIP архив (рекомендуется)
1. Скачайте `OrderManager_Portable_v1.0.zip` (251 MB)
2. Распакуйте архив в любую папку (например, `C:\OrderManager`)
3. Запустите `OrderManager.exe`

### Вариант 2: Папка напрямую
1. Скопируйте папку `OrderManager_Portable` на компьютер
2. Запустите `OrderManager.exe`

## Первый запуск

1. Запустите `OrderManager.exe`
2. База данных создастся автоматически в папке `data`
3. Настройте WebDAV в вкладке "Настройки" (если нужна синхронизация)
4. Добавьте товары в вкладке "Товары"
5. Готово к работе!

## Основные функции

### 1. Управление заказами
- Создание и редактирование заказов
- Отслеживание статусов (Новые, В работе, Доставлено)
- История заказов

### 2. Отчеты (новая функция!)
**Для кассира:**
- Сводная таблица товаров на день
- Пример: `Вода 1: 13 шт (в 3 заказах)`
- Печать и экспорт

**Для курьера:**
- Список заказов по клиентам и адресам
- Детальный состав каждого заказа
- Время, сумма, способ оплаты
- Печать и экспорт

### 3. Управление клиентами
- База клиентов с телефонами
- Несколько адресов на клиента
- История заказов клиента
- Бонусная система

### 4. Управление товарами
- Каталог товаров с фотографиями
- Категории товаров
- Цены и наличие
- Импорт из 1С (CommerceML)

### 5. Синхронизация WebDAV
- Обмен данными с мобильными устройствами
- Автоматическая синхронизация
- Импорт заказов от менеджеров
- Экспорт товаров и настроек

### 6. Карта доставок
- Визуализация заказов на карте
- Маршруты курьеров
- GPS-координаты доставки

### 7. Аналитика
- Статистика продаж
- Отчеты по клиентам
- Отчеты по товарам

## Использование отчетов

### Отчет для кассира (сборка товаров)
1. Откройте вкладку "Отчеты"
2. Выберите "Для кассира (сборка)"
3. Выберите дату и статус заказов
4. Нажмите "Обновить"
5. Распечатайте список для сборки товаров

### Отчет для курьера (доставка)
1. Откройте вкладку "Отчеты"
2. Выберите "Для курьера (доставка)"
3. Выберите дату и статус "В работе"
4. Нажмите "Обновить"
5. Распечатайте маршрутный лист

## Настройка WebDAV

1. Откройте вкладку "Настройки"
2. Введите URL WebDAV сервера
3. Укажите логин и пароль
4. Настройте интервал синхронизации
5. Нажмите "Сохранить"

## Импорт товаров из 1С

1. Меню "1С (CommerceML)" → "Импорт товаров"
2. Выберите файл `import.xml` или `offers.xml`
3. Дождитесь завершения импорта
4. Товары появятся в каталоге

## Резервное копирование

База данных находится в `data/ordermanager.db`

**Ручное копирование:**
1. Закройте приложение
2. Скопируйте файл `data/ordermanager.db`
3. Сохраните в безопасное место

**Автоматическое:**
- Резервные копии создаются в `data/backups/`
- Настраивается в меню "Файл" → "Резервная копия"

## Перенос на другой компьютер

### Способ 1: Полный перенос
1. Скопируйте всю папку `OrderManager_Portable`
2. Перенесите на другой компьютер
3. Запустите `OrderManager.exe`

### Способ 2: Только данные
1. Установите portable bundle на новом компьютере
2. Скопируйте папку `data` со старого компьютера
3. Замените папку `data` на новом компьютере
4. Запустите приложение

## Обновление

1. Сохраните папку `data` (база данных и изображения)
2. Удалите старую версию
3. Распакуйте новую версию
4. Скопируйте сохраненную папку `data` в новую версию
5. Запустите приложение

## Устранение проблем

### Приложение не запускается
- Проверьте, что файл `OrderManager.exe` не заблокирован антивирусом
- Убедитесь, что папка не находится на сетевом диске
- Запустите от имени администратора

### База данных не создается
- Проверьте права на запись в папку `data`
- Убедитесь, что диск не заполнен

### Карта не загружается
- Проверьте подключение к интернету
- Убедитесь, что брандмауэр не блокирует приложение

### Синхронизация не работает
- Проверьте настройки WebDAV
- Убедитесь, что сервер доступен
- Проверьте логин и пароль
