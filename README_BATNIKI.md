# OrderManager - Центральная система

## Быстрый старт

### 1. Установка зависимостей
Запустите `install.bat` - установит все необходимые Python пакеты

### 2. Инициализация базы данных
Запустите `init_db.bat` - создаст базу данных и таблицы

### 3. Проверка системы
Запустите `check.bat` - проверит наличие Python, зависимостей и файлов проекта

### 4. Запуск приложения
Запустите `run.bat` - запустит приложение в обычном режиме

### 5. Отладка
Запустите `run_debug.bat` - запустит приложение с выводом отладочной информации

## Сборка EXE

### Создание исполняемого файла
Запустите `build_exe.bat` - создаст standalone EXE файл в папке `dist/`

**Примечание:** Первая сборка может занять 5-10 минут

### Очистка временных файлов
Запустите `clean.bat` - удалит временные файлы сборки

## Структура батников

- `install.bat` - установка зависимостей из requirements.txt
- `init_db.bat` - инициализация базы данных
- `run.bat` - запуск приложения
- `run_debug.bat` - запуск с отладкой
- `check.bat` - проверка системы
- `build_exe.bat` - сборка в EXE
- `clean.bat` - очистка временных файлов

## Требования

- Python 3.10 или новее
- pip (обычно идет с Python)
- Windows 7 или новее

## Первый запуск

1. Запустите `install.bat`
2. Запустите `init_db.bat`
3. Запустите `check.bat` для проверки
4. Запустите `run.bat`

## Распространение

Для распространения приложения:

1. Запустите `build_exe.bat`
2. Скопируйте `dist/OrderManager.exe`
3. При необходимости скопируйте `data/central.db` (если нужна готовая БД)

Получатель сможет запустить EXE без установки Python!
