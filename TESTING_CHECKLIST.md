## 🔄 Тестирование синхронизации

### Подготовка
- [ ] Создать папку для тестового менеджера: `data/webdav/1234/outgoing/`
- [ ] Создать тестовый файл заказа (см. пример ниже)

### Ручная синхронизация
- [ ] Нажать F5 или кнопку "Синхронизировать"
- [ ] Выбрать режим "Только выгрузка"
- [ ] Нажать "Начать синхронизацию"
- [ ] Проверить лог операций
- [ ] Проверить создание файлов в `data/webdav/1234/incoming/`:
  - products.json
  - settings.json
- [ ] Открыть файлы и проверить содержимое

      "address_latitude": 55.751244,
      "address_longitude": 37.618423,

### Загрузка данных
- [ ] Выбрать режим "Только загрузка"
- [ ] Нажать "Начать синхронизацию"
- [ ] Проверить лог операций
- [ ] Открыть вкладку "Заказы" - должен появиться тестовый заказ
- [ ] Открыть вкладку "Клиенты" - должен появиться тестовый клиент
