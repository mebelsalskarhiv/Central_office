# Краткая инструкция - Вкладка "Отчеты"

## Что добавлено

✅ Новая вкладка "Отчеты" между "Заказы" и "Клиенты"
✅ Два типа отчетов: для кассира и для курьера
✅ Фильтрация по дате и статусу заказа
✅ Печать и экспорт отчетов

## Как использовать

### Для кассира (сборка товаров)

1. Откройте вкладку "Отчеты"
2. Выберите вкладку "Для кассира (сборка)"
3. Выберите дату доставки
4. Выберите статус "Новые" или "В сборке"
5. Нажмите "Обновить"
6. Получите сводную таблицу:
   ```
   Вода 1: 13 шт (в 3 заказах)
   Хлеб: 5 шт (в 2 заказах)
   ```
7. Нажмите "Печать" для распечатки

### Для курьера (доставка)

1. Откройте вкладку "Отчеты"
2. Выберите вкладку "Для курьера (доставка)"
3. Выберите дату доставки
4. Выберите статус "В доставке"
5. Нажмите "Обновить"
6. Получите список по клиентам и адресам:
   ```
   КЛИЕНТ: Дмитрий (+79281527177)
     Адрес: Сальск, Шкирятова 117
       Заказ №ORD-001
       Состав: Вода 1 (3 шт), Хлеб (2 шт)
     
     Адрес: Сальск, Ленина 25
       Заказ №ORD-002
       Состав: Вода 1 (7 шт)
   ```
7. Нажмите "Печать" для распечатки
