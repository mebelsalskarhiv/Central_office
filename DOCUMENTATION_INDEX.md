## 🔧 Батники (Batch файлы)

### Основные
- **install.bat** - Установка зависимостей Python
- **init_db.bat** - Инициализация базы данных
- **run.bat** - Запуск приложения
- **run_debug.bat** - Запуск с отладочной информацией

### Утилиты
- **check.bat** - Проверка системы и зависимостей
- **clean.bat** - Очистка временных файлов
- **build_exe.bat** - Сборка в standalone EXE
